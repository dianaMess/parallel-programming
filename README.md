# parallel-programming  
## Implementation of bubble-sorting and Gauss method by forks and pthreads  
### How to compile:  
In fork directory:  
make switch  
In pthread directory:  
make sort  
make slau
### How to run:  
for fork:  
./fork-parallel   n  
for pthreads:  
./sort   n  
./slau   n
where 'n' is array's size
